# Discord-бот, основанный на работе локальной LLM в LM Studio.
# Есть задел под использование API, в принципе - будет отдельный бранч, когда руки дотянутся.
# Only Russian language support, TBC

предусматривает openhardwaremonitor для реал-тайм отслеживания загрузки видеокарты. в остальном - все берет из-под винды. тестировалось на AMDGPU

перед запуском необходимо создать .env файл и настроить его под себя, пример - .env.example (уже закинут в репозиторий, настроить под себя)
